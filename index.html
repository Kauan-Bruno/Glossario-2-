Glossário

Primeira Questão:

1 - Social
Algo relacionado àquela marca de bolacha.
Refere-se a tudo o que está relacionado à sociedade, interação entre pessoas ou ao bem-estar coletivo.
Público; Coletivo

2 - Implementação
Algo relacionado a menta.
Ato de colocar algo em prática, executar ou dar efetivação a um plano, ideia, projeto, política ou sistema.
Estabelecer; Adicionar

3 - HTML
Linguagem de marcação usada para estruturar páginas da web.
Define títulos, parágrafos, imagens e links.
Hipertexto; Estrutura

4 - Java
Linguagem de programação orientada a objetos.
Usada em aplicativos, sistemas e jogos.
Programação; Plataforma

5 - Script
Conjunto de instruções escritas para serem executadas por um programa.
Geralmente usado para automatizar tarefas.
Código; Rotina

Segunda Questão:

1 - Exibido
Algo que é mostrado ou apresentado.
Também pode significar alguém vaidoso.
Mostrado; Ostentado

2 - Spam
Mensagens indesejadas enviadas em massa, geralmente com fins comerciais.
Comum em e-mails.
Lixo eletrônico; Propaganda

3 - Clicar
Ato de pressionar um botão do mouse ou tocar em uma tela para executar uma ação.
Selecionar; Escolher

4 - ID
Identificador único usado para diferenciar elementos ou usuários.
Identidade; Código

5 - Retornar
Voltar ao ponto anterior ou devolver um valor.
Usado em programação para indicar o resultado de uma função.
Voltar; Devolver

Terceira Questão:

1 - Cobrado
Algo que foi exigido ou solicitado em troca de algo.
Relacionado a pagamento ou responsabilidade.
Exigido; Solicitado

2 - Verifica
Confere se algo está correto ou funcionando.
Usado em lógica de programação para testar condições.
Checa; Analisa

3 - Depende
Está condicionado a outro fator ou situação.
Não é fixo, varia conforme o contexto.
Condicionado; Relativo

4 - Parâmetro
Valor passado para uma função ou método para influenciar seu comportamento.
Critério; Referência

5 - Incrementar
Aumentar o valor de algo, geralmente em uma unidade.
Usado para contadores ou repetições.
Aumentar; Somar
